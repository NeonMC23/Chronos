Polices — Inter (local)

Pour une expérience 100% hors-ligne, placez les fichiers de police Inter (format .woff2 de préférence) dans ce dossier.

Exemples de noms acceptés par la feuille de style :
- Inter-Regular.woff2
- Inter-Bold.woff2
- Inter-Variable.woff2

La feuille de style `static/css/styles.css` référence `Inter` comme famille primaire. Si vous n'avez pas Inter, le rendu utilisera la police système en fallback.
